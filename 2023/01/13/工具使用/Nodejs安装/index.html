Created2023-01-13|Updated2023-01-13|工具使用
|Post View:

简介

简单的说 Node.js 就是运行在服务端的 JavaScript。

Node.js 是一个基于 Chrome JavaScript 运行时建立的一个平台。 是一个事件驱动 I/O 服务端 JavaScript 环境，基于 Google 的 V8 引擎，V8 引擎执行 Javascript 的速度非常快，性能非常好。

下载

进入官网-Download | Node.js

LTS为对于大多数用户推荐的版本，一般来说没有特殊需求我们就直接下载这个即可；

安装

  1. 勾选协议，点击next

  1. 选择安装路径，点击next

  1. 选择安装选项，我这里选择添加到环境变量，点击next

  1. 选择是否需要组件，不需勾选，直接next

  1. 安装，install等待一会儿即可

安装完成
